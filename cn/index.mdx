autClaw 面向 Linux + Docker 生产环境。Docker 镜像提供稳定运行基座，应用本体、前端资源、插件目录和运行时依赖默认持久化到 `/opt/autclaw`。

## 常用命令

```bash
# 启动
docker run -d --name autclaw --restart unless-stopped --network bridge \
  -p 8200:8200 \
  -v /opt/autclaw:/opt/autclaw \
  nolan1121953/autclaw:latest

# 查看日志
docker logs -f autclaw

# 升级 autClaw
# 网页右上角打开“系统升级”，或在 IM 渠道发送：更新
```

<Note>
  autClaw 当前以 Linux 包单独启动和 Docker 外壳 + Linux 包启动为主，不兼容 Windows。
</Note>
