autClaw is built for Linux and Docker production environments. The Docker image provides a stable runtime base. By default, the app, frontend assets, plugin directory, and runtime dependencies are persisted under `/opt/autclaw`.

## Common commands

```bash
# Start
docker run -d --name autclaw --restart unless-stopped --network bridge \
  -p 8200:8200 \
  -v /opt/autclaw:/opt/autclaw \
  nolan1121953/autclaw:latest

# View logs
docker logs -f autclaw

# Upgrade autClaw
# Open "System upgrade" in the top-right corner of the web app,
# or send this command in an IM channel: update
```

<Note>
  autClaw currently targets Linux package execution and Docker + Linux package execution. Windows is not supported.
</Note>
